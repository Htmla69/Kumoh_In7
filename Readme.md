### 운영목적
자료실은 아래 이유에 근거해 운영됩니다.  
**소멸되고 제거되는 각종 정보들을 문서화하여 정보의 비휘발성을 높인다.**  
**분산되어 있는 정보를 종합하여 정보의 접근성과 생산성을 높인다.**  
**학생 편의를 증진하고 학교 발전을 도모한다.**
